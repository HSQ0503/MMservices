# MM Services & Cleaning

Professional website for **MM Services & Cleaning**, a local home services company in Orlando, FL with 15+ years of experience.

## Services

- Pressure Washing
- House Cleaning
- Painting Services
- Handyman & Repairs
- HVAC & Dryer Duct Cleaning
- Exterior Maintenance

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — scroll animations & micro-interactions
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
app/
├── layout.tsx          Root layout (nav + footer)
├── page.tsx            Home page
├── about/page.tsx      About page
├── contact/page.tsx    Contact page
├── gallery/page.tsx    Before/after gallery
└── services/page.tsx   Services page
components/             Reusable UI components
lib/constants.ts        Company data, services, testimonials
public/
├── logo.png            Company logo
├── Hero.jpg            Hero background image
├── gallery/            Before/after project photos (1-12)
└── services/           Service category images
```
